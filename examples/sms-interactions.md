## SMS Features

тЬЕ **Instant scam checking** - Quick number lookup  
тЬЕ **No call needed** - Text-based help  
тЬЕ **Multilingual** - Auto-detects language  
тЬЕ **Quick commands** - HELP, STATUS, CHECK, BLOCK  
тЬЕ **Escalation** - Suggests calling for complex tasks  
тЬЕ **Always available** - 24/7 SMS support  

## SMS Limitations

тЪая╕П **Complex tasks** - Requires phone call (Zoom, WiFi setup)  
тЪая╕П **Step-by-step guidance** - Better via voice  
тЪая╕П **Emergency situations** - Should call instead  

## When to Use SMS vs. Call

### Use SMS for:
- Quick scam checks
- Status updates
- Simple questions
- When unable to talk

### Use Call for:
- Guided tasks (Zoom, WiFi, volume)
- Complex questions
- Urgent help needed
- Prefer voice interaction
